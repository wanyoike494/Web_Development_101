Core & Arithmetic
These handle the "bread and butter" math operations.

Basic: add, subtract, multiply, divide, mod, pow.

Rounding: ceil, floor, fix, round.

Other: abs, sqrt, cbrt, exp, log, log10.

Trigonometry
All of these work with radians by default, but math.js can handle units like deg.

Standard: sin, cos, tan, sec, csc, cot.

Inverse: asin, acos, atan, atan2.

Hyperbolic: sinh, cosh, tanh.

Statistics & Probability
Perfect for analyzing datasets or calculating odds.

Descriptive: mean, median, mode, std (standard deviation), variance, min, max, sum, prod.

Probability: factorial, combinations, permutations, gamma, random.

Linear Algebra (Matrices)
Math.js is famous for its powerful matrix support.

Creation: matrix, ones, zeros, identity, range.

Operations: det (determinant), inv (inverse), transpose, dot, cross, trace.

Formatting: flatten, reshape, resize, sort.

Complex Numbers & Units
Complex: complex, re (real part), im (imaginary part), conj (conjugate), arg.

Units: unit (e.g., math.unit('5 cm')), createUnit.
